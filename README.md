# World Calendar Converter (JDN-Based System)

A modular, educational calendar conversion system that implements multiple world calendars and unifies them through a shared intermediate representation: the **Julian Day Number (JDN)**.

This project focuses on understanding how different calendar systems work by implementing them from scratch and connecting them through a common mathematical bridge.

---

# Goal

The goal of this project is to:

- Implement multiple calendar systems from scratch
- Understand their internal rules and structures deeply
- Build a unified conversion system using JDN
- Practice modular software design and algorithm implementation
- Explore real-world complexity in timekeeping systems

This is primarily a **learning-focused systems programming project**.
